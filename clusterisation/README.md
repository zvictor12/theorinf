# Clusterisation

Что делает:

- загружает датасет банковских транзакций;
- обрабатывает пропуски;
- строит признаки: возраст, час транзакции, логарифмы баланса и суммы;
- обрабатывает категориальные признаки `CustGender` и `CustLocation`;
- масштабирует данные;
- применяет PCA, реализованный вручную;
- применяет k-means, реализованный вручную;
- выводит средние значения и распределения признаков по кластерам.

## Локальный запуск

Из корня проекта:

```bash
uv sync
uv run jupyter notebook clusterisation/kmeans.ipynb
```

CSV должен лежать в:

```text
data/bank_transactions.csv
```
