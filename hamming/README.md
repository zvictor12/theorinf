# Hamming

Ноутбук: `error_correcting.ipynb`.

Что делает:

- кодирует текст Хаффманом;
- показывает энтропию и среднюю длину для блоков из `1` и `2` символов;
- защищает битовый поток кодом Хэмминга `(7,4)`;
- пропускает поток через канал с одиночными ошибками в некоторых блоках;
- восстанавливает исходный поток и текст.

## Запуск в Colab

Ноутбук рассчитан на запуск в облаке. Достаточно открыть `error_correcting.ipynb` в Colab и выполнить ячейки сверху вниз.

Текст скачивается напрямую из Project Gutenberg по URL внутри ноутбука.
