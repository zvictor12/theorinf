# Huffman

Что делает:

- загружает текст **The Brothers Karamazov**;
- строит код Хаффмана вручную;
- кодирует блоками из `1` и `2` символов;
- выводит энтропию и среднюю длину кода для каждого варианта;
- сравнивает размер сжатого потока отдельно и вместе с метаданными;
- сохраняет сжатые потоки и таблицы кодов в `data/`.

## Локальный запуск

Из корня проекта:

```bash
uv sync
uv run jupyter notebook huffman/sample_compression.ipynb
```

Текст уже должен лежать в:

```text
data/brothers_karamazov.txt
```

После запуска появятся промежуточные файлы:

```text
data/huffman_block_1.bin
data/huffman_block_1.json
data/huffman_block_2.bin
data/huffman_block_2.json
```

Файл `.bin` хранит упакованный битовый поток, а `.json` хранит минимальные метаданные для декодирования:

- размер блока;
- длину настоящего битового потока;
- таблицу кодов.
