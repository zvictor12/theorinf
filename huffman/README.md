# Huffman

Ноутбук: `sample_compression.ipynb`.

Что делает:

- загружает текст **The Brothers Karamazov**;
- строит код Хаффмана вручную;
- кодирует блоками из `1` и `2` символов;
- выводит энтропию и среднюю длину кода для каждого варианта;
- сравнивает размер сжатого потока.

## Запуск в Colab

Ноутбук рассчитан на запуск в облаке. Достаточно открыть `sample_compression.ipynb` в Colab и выполнить ячейки сверху вниз.

Текст скачивается напрямую из Project Gutenberg по URL внутри ноутбука.
